# KeyboardNinja

Simple typing tutor

## How to build

Using `nmake`:
```sh
> nmake modules
> nmake main
```

## How to test

Build tests using `nmake`:
```sh
> nmake tests
```
Use `nmake test` to run tests.

## How to run

* Use `nmake run` to run a program. 
* or run `kninja.exe`. Make sure `kninja.exe` file and `data` folder located in the same folder.
